# 왜 만들었냐?

pdf 로 된 논문을 그대로 마우스로 긁어서 복붙하면 newline 이 들어간 형태로 복붙이 된다.

이 상태에서 구글 번역기를 돌리면 매우 안좋은 품질로 번역된다.

이걸 뉴라인을 없애주면 예쁘게 번역된다.
